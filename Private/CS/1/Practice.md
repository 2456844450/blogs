# 练习题

## 1.1

![image](http://shadows-mall.oss-cn-shenzhen.aliyuncs.com/images/assets/cs/39.png)

解：

A. 
  k = 150 / 100 = 1.5;
  r = 1500 / 2500 = 0.6;
  t = (1 - 0.6) + 0.6 / 1.5 = 0.4 + 0.4 = 0.8;
  加速比 = 1 / t = 1 / 0.8 = 1.25;

  加速比为 1.25
B.
  t = 1 / 加速比 = 0.6;
  k = 0.6 / (0.6 - 0.4) = 3;
  s = s * k = 100 * 3 = 300;

  需要以 300 公里的速度通过蒙大拿州。
